# 📚 Personal Study Tracker

A simple study progress tracker built using HTML, CSS, and JavaScript.

## 🚀 Features
- Displays subjects and progress percentage
- Calculates average progress
- Clean UI with background styling

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- DOM Manipulation

## 📂 Project Structure
- tracker.html
- tracker.css
- tracker.js

## 🎯 Future Improvements
- Add new subject dynamically
- Progress bar UI
- Save data using Local Storage
